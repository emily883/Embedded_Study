# Embedded Study

[Version en espanol](./README.md)

This repository is my space to study embedded systems at my own pace, collect useful resources, and write down what I learn along the way.

I do not want to rush this process or skip the basics. The point is to learn things properly, really understand what I am studying, and slowly build a solid foundation.

## What I want to do here

I want to use this repo as a kind of study log. This is where I will keep:

- books and resources
- notes
- examples
- practice work
- things I want to come back to later

## Study notes

- I installed GCC so I could start compiling C examples from the terminal.
- The version I installed was `gcc 15.2.0`.
- I installed it with `winget`, using the package `BrechtSanders.WinLibs.POSIX.UCRT`.
- To compile one of the examples, I used this command:

```bash
gcc -std=c2x -Wall -lm -o getting-started getting-started.c
```
